КОНТРОЛЬНАЯ РАБОТА: PYTHON

ВАРИАНТ 1

ИНСТРУКЦИЯ ПО ВЫПОЛНЕНИЮ:

> Время выполнения: 40 минут

> Запрещено использовать интернет и готовые решения

> Каждая задача должна быть полностью решена и протестирована

ЗАДАЧА 1
БАЗОВЫЙ УРОВЕНЬ

ПРОВЕРКА ЧИСЛА

Напишите программу, которая запрашивает число и выводит:

  • "Положительное" если число > 0
  • "Отрицательное" если число < 0
  • "Ноль" если число = 0
ЗАДАЧА 2
БАЗОВЫЙ УРОВЕНЬ

ПРОВЕРКА ПАРОЛЯ

Напишите программу, которая проверяет пароль:

  • Если пароль "12345", вывести "Доступ разрешен"
  • Иначе вывести "Доступ запрещен"
ЗАДАЧА 3
БАЗОВЫЙ УРОВЕНЬ

КАТЕГОРИЯ ВОЗРАСТА

Напишите программу, которая определяет категорию по возрасту:

  • До 7 лет: "Дошкольник"
  • 7-17 лет: "Школьник"
  • 18+ лет: "Взрослый"
ЗАДАЧА 4
БАЗОВЫЙ УРОВЕНЬ

ПРОВЕРКА ЧЕТНОСТИ

Напишите программу, которая проверяет число на четность:

  • Если число четное, вывести "Четное"
  • Если число нечетное, вывести "Нечетное"
ЗАДАЧА 5
БАЗОВЫЙ УРОВЕНЬ

ПРОСТОЙ КАЛЬКУЛЯТОР

Напишите программу-калькулятор для двух чисел:

  • Пользователь вводит два числа и операцию (+, -, *, /)
  • Программа выводит результат
  • При делении на ноль вывести "Ошибка"

КРИТЕРИИ ОЦЕНКИ:

Оценка "3" - решены 3 задачи

Оценка "4" - решены 4 задачи

Оценка "5" - решены все 5 задач
